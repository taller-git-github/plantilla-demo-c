# Tarea ejemplo de programación en C

## Problema #1: suma de dos números enteros
Escribe un programa en C para calcular la suma de dos números enteros.

### Ejemplo

#### Entrada:
```
3 5
```

#### Salida:
```
8
```

#### Pruebas
Ejecutar desde la terminal el comando:
```
$ make suma
```
